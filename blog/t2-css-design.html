Borders, Margins, & Padding

Understanding the Difference

11/6/14

When using html and CSS, every item or element that you create (be it a heading, paragraph, table, anything) is surrounded by an element box.This box consists of three different layers that you can manipulate using CSS.

  • The first, outer most layer of this element box is the margin. The margin determines where that box will sit on the page as well as how closely that box will sit next to other boxes.

  • The next layer is the border. The border will outlines the padding and contexts of the box. When it's thickness is increased, the border will expand until it the margin limitations on all four sides. At that time it will start to encroach on the padding.

  • The last layer is the padding. The padding determines how much space there is between the content of the box and the border. Similarly to the border, when you increase the size of the padding it will expand the box until it reaches the limitations defined by the margins at which time it will start to encroach on the content space.

All three of these layers have four different variables that you can set. A top, right, bottom, and left. You can set all of these to the same value by using the commands margin, border, or padding as so:
  • margin: 10px;
  • border: 1px;
  • padding: 5px;
You can also set each side individually using the commands (layer)-top/right/bottom/left as so:
  • margin-bottom: 50px
  • border-top: 1%
